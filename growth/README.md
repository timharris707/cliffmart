# Cliff Circuit Growth System

## Components
- `upstream-targets.json` — accounts to engage with daily
- `daily-routine.md` — 25-touch engagement checklist
- `reply-templates.md` — swipe file for fast replies
- `thread-calendar.md` — schedule for value threads + recaps
- `signal-group.md` — micro signal group plan
- `weekly-review.sh` — template for Friday analytics report

## Daily Flow
1. Run through `daily-routine.md`
2. Log learnings in `logs/daily-engagement-YYYY-MM-DD.md`
3. Save notable tweets to swipe file

## Weekly Flow
- Monday/Thursday: publish value threads
- Sunday: week-in-public recap
- Friday: run `growth/weekly-review.sh`

## Metrics to Track
- Followers gained per week
- Average engagement per tweet
- Number of inbound DMs/collabs
