# Cliff Circuit Daily Growth Routine

**Goal:** 25+ meaningful interactions per day (5 threads × 4 engagement touches + misc likes)

## Morning Checklist (15 minutes)
- [ ] Scan upstream-targets list
- [ ] Turn on notifications for top 5 accounts
- [ ] Check DMs/mentions

## Engagement Loop (repeat 5 times)
1. [ ] Find fresh thread (< 1 hour old)
2. [ ] Like post
3. [ ] Add thoughtful reply (use templates)
4. [ ] Quote tweet if relevant
5. [ ] Drop a resource/link only if additive

## Afternoon Touches
- [ ] Reply to comments on our posts
- [ ] DM 1 interesting account
- [ ] Save best-performing tweets to swipe file

## Evening Wrap
- [ ] Review analytics dashboard
- [ ] Note learnings in `logs/daily-engagement-YYYY-MM-DD.md`
